Seus dados estão seguros?

Seus dados ficam salvos apenas no navegador deste dispositivo. Se você limpar os dados do navegador (histórico, cookies, cache), todas as suas informações serão perdidas.

Para evitar isso, vá em Configurações → Backup e Restauração e faça o download do arquivo .json. Guarde este arquivo em um local seguro.

Atenção: este arquivo contém seus dados pessoais (login, senha e informações financeiras). Não compartilhe com ninguém!

Para restaurar, use o mesmo arquivo em Configurações → Restaurar.
